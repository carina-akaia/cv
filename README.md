## Getting started

### Install runtime manager

<https://github.com/jdx/rtx#quickstart>

### Install Basic dependencies

```sh
rtx trust && rtx i && cargo install cargo-make
```

### Install advanced development tools ( optional )

```sh
cargo install cargo-modules
```

## Development server

```sh
cargo make dev
```

## Production build

```sh
cargo make build
```
